react컴포넌트 typescript생성으로 해보기

2020-01-17

서론


npx create-react-app 를 통해 생성한 typescript 기반의 리액트 프로젝트를 살펴보면서 리액트를 연구하고 있습니다.

리액트 컴포넌트 생성시 필요한 유용한 정보에 대해 알아보겠습니다. 

 

연구내용

컴포넌트에 생략 할 수 있는 Props 설정하기

컴포넌트 props 값 중에서 생략해도 되는 값이 있다면 ? 문자를 사용하면 됩니다.  아래 Greetings 컴포넌트는 optional 값이 존재하는 경우 optional 값을 추가로 출력하게 됩니다. 

import React from 'react';
 
type GreetingsProps = {
  name: string;
  mark: string;
  optional?: string;
};
 
function Greetings({ name, mark, optional }: GreetingsProps) {
  return (
    <div>
      Hello, {name} {mark}
      {optional && <p>{optional}</p>}
    </div>
  );
}
 
Greetings.defaultProps = {
  mark: '!'
};
 
export default Greetings;


app.tsx 파일에서 props 값으로 옵션값을 전달하지 않는 경우, 에러 발생 없이 출력되는 것을 확인 할 수 있습니다. 

const App: React.FC = () => {
    return <Greetings name="Hello"/>;
};


app.tsx 파일에서 props 값으로 옵션값을 전달하는 경우에도, 에러 발생 없이 출력되는 것을 확인 할 수 있습니다. 

const App: React.FC = () => {
    return <Greetings name="Hello" optional="옵션"/>;
};


컴포넌트에서 함수 타입의 props 받아오기 

특정함수를 props 로 받아오기 위해서는 타입을 지정해야 합니다. 

import React from 'react';
 
type GreetingsProps = {
    name: string;
    mark: string;
    optional?: string;
    onClick: (name: string) => void; // 아무것도 리턴하지 않는다는 함수를 의미합니다.
};
 
function Greetings({ name, mark, optional, onClick }: GreetingsProps) {
    const handleClick = () => onClick(name);
    return (
        <div>
            Hello, {name} {mark}
            {optional && <p>{optional}</p>}
            <div>
                <button onClick={handleClick}>Click Me</button>
            </div>
        </div>
    );
}
 
Greetings.defaultProps = {
    mark: '!'
};
 
export default Greetings;


app.tsx파일에서는 다음과 같이 작성합니다.  참고한 자료에서 클릭 이벤트를 한번에 파악할 수 있도록 코드를 살짝 수정했습니다.

onclick 이벤트가 발생하게 되면 alert으로 name을 출력하게 됩니다.  

import React from 'react';
import Greetings from './Greetings';
 
const App: React.FC = () => {
    const onClick = (name: string) => {
       alert(name);
    };
    return <Greetings name="Hello" onClick={onClick} />;
};
 
export default App;




결론

리액트 컴포넌트 생성시 생략할 수 있는 props 설정 방법, 함수 형태의 props 받아오는 방법에 대해서 연구하였습니다. 

생략할 수 있는 props 값을 설정하기 위해서는 ? 를 사용하며,  함수 형태의 props를 받아오기 위해서는 props 선언시 타입을 지정해주게 됩니다.
